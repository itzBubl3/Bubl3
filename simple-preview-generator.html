Générateur d'Image de Prévisualisation pour Réseaux Sociaux

Cet outil va générer une image de 1200x630px optimisée pour Discord, Twitter et autres réseaux sociaux.

L'image sera utilisée quand vous partagez votre lien : https://itzbubl3.github.io/Bubl3/

Instructions :

  1. Cliquez sur "Générer l'Image" pour créer la prévisualisation
  2. Cliquez sur "Télécharger" pour sauvegarder l'image
  3. Uploadez l'image sur votre site GitHub Pages
  4. Le nom de fichier doit être : preview-image.png
